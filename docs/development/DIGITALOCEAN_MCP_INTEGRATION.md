minimum required permissions\n- **Audit access** logs monthly\n\n### Network Security\n- Enable **firewalls** on all droplets\n- Use **VPC** for internal communication\n- Implement **SSL/TLS** everywhere\n- **Monitor** for suspicious activity\n\n### Access Control\n- **Team-based** access to DigitalOcean console\n- **Role-based** permissions for MCP actions\n- **Multi-factor authentication** required\n- **Audit trails** for all deployments\n\n---\n\n## Cost Optimization\n\n### Resource Sizing\n- **App Platform**: Start with `basic-xxs` ($5/month)\n- **Database**: Use
